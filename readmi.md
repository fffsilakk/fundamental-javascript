# 🚀 JavaScript Fundamental Course - 2026 Edition

Repositori ini berisi kumpulan materi dasar hingga menengah untuk mempelajari JavaScript modern (ES6+). Cocok untuk pemula yang ingin memahami konsep pemrograman dari nol hingga mampu membuat logika aplikasi sederhana.

## 📚 Daftar Materi

Repositori ini dibagi menjadi beberapa modul untuk mempermudah proses belajar:

1.  **Variables & Scope** (`01-variables.js`): Memahami perbedaan `var`, `let`, dan `const` serta konsep _Hoisting_.
2.  **Data Types** (`02-data-types.js`): Mengenal tipe data _Primitive_ (String, Number, BigInt, dll) dan _Reference_.
3.  **Functions Deep Dive** (`03-functions.js`): Belajar _Arrow Function_, _Callback_, hingga _Closures_.
4.  **Control Flow** (`04-control-flow.js`): Logika pengkondisian (_If-Else, Switch, Ternary_) dan perulangan modern.
5.  **Array & Object** (`05-arrays-objects.js`): Manipulasi data kompleks, _Destructuring_, dan _Spread Operator_.
6.  **Arithmetic Operators** (`09-arithmetic-operators.js`): Operasi matematika, _Increment_, dan _Math Object_.
